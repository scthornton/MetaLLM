# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- 40+ exploit modules covering OWASP LLM Top 10 vulnerabilities
- 15+ auxiliary modules for reconnaissance and assessment
- Interactive CLI with Metasploit-style interface
- Modular architecture for easy extension
- Comprehensive coverage of LLM, RAG, Agent, MLOps, and API security
- Structured logging and detailed reporting
- Target management system
- Session persistence
- Exploit and auxiliary module base classes
- Complete documentation and user guides

### Security
- Ethical use guidelines and warnings
- Authorized testing requirements enforcement
- No hardcoded credentials or API keys
- Comprehensive .gitignore for sensitive data

## [1.0.0-alpha] - 2026-01-20

### Added
- Initial alpha release of MetaLLM Framework
- Core framework engine with module loader
- 40+ exploit modules across LLM, RAG, Agent, MLOps, and API categories
- 15+ auxiliary modules for scanning, fingerprinting, and discovery
- Interactive console interface
- Configuration management system
- Target and session management
- Comprehensive documentation (Architecture, CLI Guide, Module Library)
- Example workflows and use cases
- GCP test lab infrastructure templates
